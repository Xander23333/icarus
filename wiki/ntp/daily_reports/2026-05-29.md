## 1. 今日新增重要论文

### 1. [arXiv:2605.26379] When Does LeJEPA Learn a World Model?
- **核心问题**：JEPA 一族能否在数学上 *线性* 恢复世界 latent？Gaussian 先验是必须的吗？
- **方法**：在 stationary additive-noise 世界类上分析 LeJEPA = alignment + Gaussian reg；证 linear identifiability 正面定理 + Gaussian 唯一性反面定理 + approximate identifiability 退化界；2D→1024d→pixel 控制实验。
- **核心结论**：在该世界类下 alignment + Gaussian reg ⇒ linear identifiability；Gaussian 是 *唯一* satisfying prior；linear-orthogonal identifiability ⇒ optimal latent planning。
- **NTP 归类**：NTP-mech (paradigm-replacement 路线条件性数学加固)。
- **与已有工作的关系**：与 [2605.23972] LDI/Flux 形成理论-实证配对；与 [2605.26900] SPHERE-JEPA 形成几何先验之争（manifold 下 Gaussian 失败）；与 §10 C1 正交，与 C6 video-NTP candidate 互补。
- **是否提出新理论边界**：yes — *在 stationary additive-noise 世界类条件下* LeJEPA linear identifiability + Gaussian 唯一性双定理。

### 2. [arXiv:2605.26362] Why LLMs Hallucinate on Structured Knowledge: A Mechanistic Analysis
- **核心问题**：linearized graph/table/multi-hop context 充分时 LLM 仍幻觉，机制定位 attention 还是 FFN？跨结构能否通用？
- **方法**：在 graph/multi-hop/table 三种 schema 上做 attention entropy + activation patching + FFN probing，逐层定位 grounding 失败；构造跨 schema hallucination 检测器。
- **核心结论**：FFN 对 in-context 事实 *fail to ground* 是跨 schema 通用的 sufficient statistic；attention 集中是 task-dependent 次要 signal；detector AUC>0.8。
- **NTP 归类**：NTP-mech (FFN-grounding 层；新候选 confound 第六项)。
- **与已有工作的关系**：与 readout-side 清单 (2605.10799/07120/14004/05438/25891/11467) 同层补一条；与 N3 §5 unfaithful CoT 直接相关；与 [2605.18732] Predictable Confabulations 形成 in-context 与 in-parameter recall 的上下游。
- **是否提出新理论边界**：no — 但提出可证伪机制命题 "FFN-grounding 是 sufficient statistic"。

### 3. [arXiv:2605.26733] STARS — Stability-driven Recurrent Scaling for LoopLMs
- **核心问题**：LoopLM 在 test-time 加深为何崩塌？是 latent reasoning 的根本不稳定还是训练目标缺约束？
- **方法**：谱分析 LoopLM 的 Jacobian；提 Jacobian Spectral Radius Regularization + 随机 loop 深度采样训练，把 latent map 约束到 asymptotically stable fixed point。
- **核心结论**：STARS 把 Jacobian 谱半径压到 <1，test-time depth 单调饱和或继续提升；算术 + 复杂数学 reasoning 同时改善 peak。
- **NTP 归类**：counter-evidence（"latent reasoning 不稳定 = NTP mech 缺陷"被修复反证）；同时迫使 C1 参数集添 *convergence-mode*。
- **与已有工作的关系**：与今日 [2605.26379] 在 *latent 空间该满足什么结构性约束* 维度并列（一动力学侧，一分布侧）；扩 [2605.24956]/[2605.05438]/[2605.11467]/[2605.14004] objective-engineering 路线到第 4 条 lever；与 N2 (TC⁰ wall) 在 token-CoT vs latent-loop 双轨上互补。
- **是否提出新理论边界**：no formal — 隐含设计原则 "test-time scalable latent reasoning ⟺ trained-stable latent map"。

### 4. [arXiv:2605.26820] Can VLA Models Learn Real-World Data Continually without Forgetting?
- **核心问题**：把 VLA continual learning 从模拟换到真实异构机器人 demo，遗忘是否仍可控？
- **方法**：构 4 任务真实 sequential dataset（rigid pick-place、contact-rich pressing、deformable folding）；扫常见 VLA backbone 与 experience-replay 实现细节。
- **核心结论**：catastrophic forgetting 比模拟报告 *更严重*；replay 实现细节图给出但 best replay 仍未达 joint upper-bound。
- **NTP 归类**：NTP-mech (embodiment + continual 子带；C5 候选的首条公开真实 anchor)。
- **与已有工作的关系**：与昨日 [2605.25889] (C4 VLA IT 静态上界) 在 VLA 子带形成 "**静 C4 + 动 C5**" 双锚；与 N5 §3 (RT-1→RT-2→OpenVLA chronology) 把 *online updating* 阶段经验上截住；与 C-CONT-4 ripple-effect 暗线同假设家族。
- **是否提出新理论边界**：no — 但 *首条公开真实 reference*，后续 C5 否证必须超过本文 best replay。

### 5. [arXiv:2605.26655] Why Prompt Optimization Works, and Why It Sometimes Doesn't (causal-inspired edit-level)
- **核心问题**：DSpy/TextGrad 优化结果跨任务/跨 backbone 为何不传递？
- **方法**：propensity-adjusted associational analysis × 多 edit representation 三角化（cognitive-load 标注、surface 特征、edit motif）跨多框架 × backbone × benchmark。
- **核心结论**：complexity-increasing / meta-instructional edits 对 math/multi-hop reasoning 负相关；step-by-step / meta-cognitive edits 对 logical/sequential reasoning 正相关；跨 representation 一致；prompt-opt 失败是 edit-family × task-routing 系统交互。
- **NTP 归类**：counter-evidence（任何"prompt 改动暴露/掩盖 模型能力"的 mech 声称必须报 propensity-adjusted edit-family conditional effect）。
- **与已有工作的关系**：补 readout-side 清单第 7 项；与 [2605.18079] low-prec CoT、[2605.10799] format confound 同一方法论原则的 prompt 侧分摊；与今日 [2605.26362] FFN-grounding 是 *接口效应 vs 核心机制* 的另一侧。
- **是否提出新理论边界**：no — 但 *方法论命题*: "prompt-only 效应必须报 edit-family conditional effect 才可作 mech 证据"。

## 2. 重要观点摘要

- **confound 清单 5 → 7**：前 5 项 (format / verbal-readout / CE-collapse / attribute-head / representation-geometry) 是 *结构/训练* 侧；新加 (vi) FFN-grounding bottleneck、(vii) prompt edit-family × task-routing 是 *接口/方法学* 侧。"NTP 推理 mech 不足"的声称从今天起必须七项同时排除。
- **C1 (Deterministic Horizon) 第 4 参数**：STARS 证明 trained-stable looped 模型把 fixed-depth forward-pass 假设打破，C1 参数集应从 (L, d, tokenization) 升级到 **(L, d, tokenization, convergence-mode)**——同一原理在 fixed-depth NTP 上紧致，在 trained-stable looped NTP 上需以 *converged loop depth* 替代。
- **paradigm-replacement 路线获条件性数学加固，但只在 *理想化* 世界类**：LeJEPA 双定理 + SPHERE-JEPA 流形修订 合起来表明 "JEPA-类 SSL 的最优 prior 由世界几何/分布假设决定"——不能作为 "JEPA > NTP 通用结论"，但可作为 N5/N6 中 LeCun-bet 叙事的局部理论后盾。
- **VLA 子带 mech 双锚到位**：C4 (capability+robustness 静态 IT 上界) + C5 (real-world continual forgetting) 在同一子带形成静/动双重 conditional mech 候选。VLA 子带变成 §10 候选最 *形式化* 的 mech 区。
- **objective-engineering 路线第 4 条修补 lever**：STARS Jacobian-radius reg 加入 (NITP, Semantic-Loss, Conditional-Attr, ProFIL) 之后，连续 4 条独立 *与 NTP 范式相容* 的修补成立，paradigm-replacement 路线相对优势进一步收缩到 *世界几何假设强成立* 的子集。

## 4. 新出现的研究趋势

1. **接口层 vs 核心机制层 的方法论分摊**：今日 FFN-grounding (mid-layer) + prompt edit-family (input layer) 把 readout-side 主导假设从 *output 侧* 扩到 *full-stack 接口侧*。下一步可预期 attention-pattern 侧的同型工作 (mid-layer routing × task-family 交互)。
2. **latent reasoning 的 *动力学* 框架兴起**：STARS 把 LoopLM 的 stability 形式化为 Jacobian 谱半径；预计 6–12 个月内出现 "latent ODE / fixed-point reasoning" 子赛道，与 token-CoT 形成 *双轨* test-time scaling。
3. **JEPA / paradigm-replacement 路线开始 *条件化***：LeJEPA-identifiability 与 SPHERE-JEPA 联读后，研究问题从 "JEPA 是不是好" 转向 "在什么世界几何/分布假设下哪种 JEPA 变体最优"。这是该路线进入 *精细化* 阶段的信号。
4. **VLA 子带 mech 论证 *闭环化***：C4 静态 IT bound + C5 真实 continual benchmark + 已有 OpenVLA 经验数据形成 \"静上界 + 动 benchmark + 实测\" 三轨；任何 VLA-as-AGI-stepping-stone 声称必须穿越这三轨。

## 5. 当前仍未解决的问题 / 新争议

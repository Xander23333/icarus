## 1. 今日新增重要论文

### 1. [arXiv:2605.22471] Lost in Tokenization: Fundamental Trade-offs in Graph Tokenization for Transformers
- **核心问题**：transformer 表达力是否由架构单独决定？
- **方法**：对 spectral / random-walk / adjacency 三种 building-block tokenization，在一组规范图函数上给紧 depth bound。
- **核心结论**：同一函数在不同 tokenization 下 depth 之间存在**多项式级 gap**；表达力 = (architecture, tokenization) 联合属性。
- **NTP 归类**：NTP-mech (extension of C1)。
- **是否提出新理论边界**：yes — cross-tokenization depth separation。

### 2. [arXiv:2605.20271] MHA as Ensemble Nadaraya-Watson Estimation
- **核心问题**：MHA 性能增益的来源——更多 heads 还是 head 去相关？
- **方法**：用 attention ≡ NW kernel 等价把 MHA 分解为 H 个 NW 估计器的结构化 ensemble，导出 bias-variance-covariance 分解。
- **核心结论**：variance reduction 由 principal angles 主导，不由 H 单调控制；正交投影上界给最优常数。
- **NTP 归类**：NTP-mech (soft, framework)；不立新 impossibility，但为后续 expressivity / sample complexity bound 提供工具。

### 3. [arXiv:2605.16758] Language Acquisition Device in Large Language Models
- **核心问题**：LLM 数据效率 gap 能否用 LAD-style 限制性 prior 缩短？
- **方法**：合成形式语言 MP-STRUCT (MERGE/AGREE/MOVE)；500 步 PPT vs k-Shuffle Dyck 5k+ 步 baseline。
- **核心结论**：表达力**更弱**的 prior 在数据效率上**更优**；BLiMP 长程子集反超。
- **NTP 归类**：NTP-cap — 削弱"sample-efficiency gap = NTP 机制问题"的论据。

### 4. [arXiv:2605.14004] Conditional Attribute Estimation with Autoregressive Sequence Models
- **核心问题**：NTP local objective 是否天然不擅长 sequence-level attribute 估计与控制？
- **方法**：双 head (next-token, attribute) 联合训练；对比 PPLM / FUDGE / post-hoc classifier。
- **核心结论**：方差与计算成本同时降一个数量级；说明"NTP 不会全局"中一部分是 readout/objective 工程问题，不是机制。
- **NTP 归类**：NTP-cap，弱化 mech 论据。

### 5. [arXiv:2605.07120] When Symbol Names Should Not Matter: A Logistic Theory of Fresh-Symbol Classification
- **核心问题**：fixed-label template 任务上 transformer 抽象 vs 名称偏置能否给 closed-form 分解？
- **方法**：transformer-kernel regime 下做正则化 kernel logistic 分析；分解 ideal classifier + vocabulary overlap perturbation。
- **核心结论**：分解给出 (V, ρ, m) 显式收敛率；**NTP 设定下分解失效**——readout 强制 token-name 依赖。
- **NTP 归类**：NTP-mech (边界澄清)。

### 6. [arXiv:2605.05438] Semantic Loss Fine-Tuning Prevents Causal-Reasoning Model Collapse
- **核心问题**：小模型 causal FT 是否系统性 collapse 但被 aggregate accuracy 掩盖？
- **方法**：Gemma 270M 在 transitivity / d-separation 上做纯 CE FT 与 semantic-loss FT 对比。
- **核心结论**：纯 CE 100% collapse、acc 73.9% 假阳性；semantic loss + 动态 λ 消除 collapse。
- **NTP 归类**：counter-evidence — 削弱"NTP 学不到因果"经验论据。

## 2. 重要观点摘要

- **Tokenization 是 NTP-mech 边界条件**：Lost in Tokenization 把"transformer 表达力"从 (L, d) 重写为 (L, d, tokenization)；BPE 对 LLM 之于现实任务很可能是某些"机制级失败"的真因，可被换 tokenizer 缓解。这把 C1 (Deterministic Horizon) 的紧度区间收窄。
- **Readout-side 主导假设**：本周累计 4 篇 (今日 2605.07120 / 2605.14004 / 2605.05438 + 昨日 2605.10799) 独立指向同一方向——多数被解读为"NTP 推理机制不足"的现象实际由 readout / format / objective / collapse 主导。下一阶段 NTP-mech 候选必须显式控制这四个 confound。
- **Prior > 表达力**：LAD-PPT 反直觉地说明限制性 prior 比强表达 prior 更高效；进一步削弱"sample-efficiency gap = NTP 机制问题"的旧论据。
- **MHA 的统计 ensemble 解释**：MHA-as-NW-Ensemble 给 attention 头多样性的紧分析提供 framework，是未来 attention 表达力上下界的工具，但暂未触及现代深层 stack。

## 3. 与 NTP 的关联分析

- **NTP-mech**：C1 (Deterministic Horizon) 参数集扩为 (L, d, tokenization)，紧度区间因此收窄；MHA-as-NW-Ensemble 提供 framework；Fresh-Symbol 给"NTP readout 强制 token-name 依赖"硬结论。
- **NTP-cap**：LAD-PPT 与 Conditional Attribute Transformers 两条 (sample-efficiency / global-attribute) 工程证据共同削弱旧 mech 论据弹药库。
- **counter-evidence**：Semantic-Loss Anti-Collapse 增强本周"方法论纠偏潮"；与 2605.10799 形成对位。
- **净效应**：NTP-mech 阵营今天**进一步收窄**（tokenization confound 把 C1 适用 setting 进一步约束），同时 readout 主导假设吸收掉多条 mech 弹药；NTP-cap 阵营有 sample-efficiency 与 attribute-head 两条边界细化。

## 4. 新出现的研究趋势

1. **Tokenization-aware expressivity**：把 tokenization 显式纳入 transformer depth/width bound 的研究开始独立成簇（图侧已成型，NLP 侧 BPE-vs-character 是顺理推论）。
2. **Readout/format meta-methodology**：CoT-faithfulness、causal FT、global-attribute 三类实验都被发现含 readout 类 confound——预计未来 6 个月有大批复验/撤回。
3. **Prior-engineering 路线**复活（LAD-PPT、合成形式语言 PPT）：从"加数据"转向"加结构性 prior"作为 sample-efficiency 主战场。
4. **Attention-as-kernel** 系列工作回潮，作为现代 architecture 统计分析的通用语言。

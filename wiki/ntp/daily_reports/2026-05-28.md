## 1. 今日新增重要论文

### 1. [arXiv:2605.24956] NITP: Next Implicit Token Prediction for LLM Pre-training
- **核心问题**：NTP one-hot 监督是否导致 latent 表征 degenerate/各向异性？是机制级缺陷还是 objective-engineering 层可救？
- **方法**：在标准 NTP 之外叠加一项辅助——用同模型浅层激活作为下一 token 的稠密 self-supervised target。
- **核心结论**：9B MoE MMLU-Pro +5.7%、C3 +6.4%、CSQA +4.3%，开销可忽略；论证收益来自 hidden-state 自由度的几何收紧。
- **NTP 归类**：counter-evidence — 把"NTP 几何欠约束 → mech 缺陷"降级为可被廉价 auxiliary 修掉的 objective-engineering 问题。
- **与已有工作的关系**：与 2605.07120 (Fresh-Symbol) / 2605.14004 (Conditional-Attr) 形成 readout/objective 三连击。
- **是否提出新理论边界**：no。

### 2. [arXiv:2605.25891] Causal Tongue-Tie: LLMs Can Encode Causal Direction, But Their Yes/No Outputs Fail to Express
- **核心问题**：LLM 在 anti-commonsense causal 题低 yes/no 准确率，是"无 causal reasoning"还是 verbal interface 与内部表征解耦？
- **方法**：固定 linear probe 解码 hidden state，对比同模型的 yes/no 输出；在 CLadder anti-commonsense 子集上量化"Causal Tongue-Tie" gap。
- **核心结论**：probe 0.97 / yes-no 0.5；≈+0.47 全部归因 verbal-readout 失败。
- **NTP 归类**：counter-evidence（削弱 yes/no 准确率作 mech proxy 的所有论据）。
- **与已有工作的关系**：与 2605.07120 (NTP readout 强制 token-name 依赖) 互为理论+实证双轨。
- **是否提出新理论边界**：no，但给出可证伪方法论命题（双轨证据要求）。

### 3. [arXiv:2605.25889] Capability and Robustness Cannot Both Be Free: An IT Bound for VLA Models
- **核心问题**：VLA 模型 clean accuracy 与 adversarial robustness 之间是否存在 policy-independent 理论上界？
- **方法**：双 DPI + MI 非负，证明 capability MI + robustness MI ≤ H(task) + adversarial channel capacity；引入 encoder-specific corollary 收紧 channel。
- **核心结论**：OpenVLA 上 pixel-bound ~10³ nats（vacuous），encoder-specific bound ~31 nats（actionable）且 policy 已吃满。
- **NTP 归类**：NTP-mech 候选（VLA 子带；discrete-action 限定）。写入 §10 C4。
- **与已有工作的关系**：补 embodiment 共识区"经验 not robust"的理论靠山；与 C1 (forward-pass capacity) 维度正交。
- **是否提出新理论边界**：yes — `cap + rob ≤ H(task) + C(adv)`。

### 4. [arXiv:2605.11467] Drop the Act: Probe-Filtered RL for Faithful CoT Reasoning (ProFIL)
- **核心问题**：能否在不被 RL 闭环吃掉的前提下，可靠惩罚 CoT 中的 post-commitment theater 步骤？
- **方法**：在 frozen base 上训练 attention probe 标 post-commitment，GRPO rollouts 中超阈值步骤的 advantage 置零（drop-in 扩展 GRPO）。
- **核心结论**：四域两架构上 CoT theater 减 11–100%，prior 预测的 RL-obfuscation 失败模式没出现。
- **NTP 归类**：counter-evidence — "CoT theater = mech 缺陷"被工程修复反证。
- **与已有工作的关系**：是 Readout-side 主导假设的首条 *干预侧* 证据，与 2605.10799/07120/14004/05438/25891 的观察侧形成闭环。
- **是否提出新理论边界**：no — 但隐含"frozen probe 不被 RL 闭环吃掉"的稳定性命题，反例 prior 失败预测。

### 5. [arXiv:2605.23972] Why We Need World Models for AGI: LDI & Flux
- **核心问题**：LLM 的 long-horizon planning 限制是 capacity 不足还是 objective-level 错配 (sequence prediction ≠ latent dynamics inference)？
- **方法**：构造 Flux（rule-defined sequential reasoning 环境）并把规则编译为 latent-state simulator；纯文本 LLM 对比在 latent state 上训的 RL agent。
- **核心结论**：latent-state RL agent 在 long-horizon 子任务上系统性优于纯文本 LLM，差距随 horizon 非线性放大且不与 context budget 同步。
- **NTP 归类**：NTP-mech 候选（弱版本，框架级）。
- **与已有工作的关系**：与 C1 正交；与 2605.05438/14004 的 "objective 工程层可救" 形成张力，阈值在 horizon 与是否允许 auxiliary head。
- **是否提出新理论边界**：no formal — 提出 LDI 概念框架。

- **NTP-mech 通用候选今天没有增加**：所有今日 mech 信号要么落 VLA 子带（C4），要么是框架级弱主张（LDI）。"通用 NTP-mech" 阵营在 readout-side 闭环冲洗后实质上只剩 C1 (no-CoT)、Reversal Curse、Ω(H) 下界三条。

## 4. 新出现的研究趋势

1. **probe-based 干预闭环**：Causal Tongue-Tie 观察 + ProFIL 干预 + 2605.07120 形式分析三轨同时上线，预示未来 6–12 个月"probe 训练 → RL 反馈"的 instrumentation 范式成主流。
2. **objective-engineering 路线**：用浅层 self-target / attribute head / semantic loss 给 NTP 加 *与范式相容* 的稠密信号，正在分化为独立子赛道，与 paradigm-replacement (JEPA/world-model) 路线形成对照。
3. **VLA 信息论上界**：2605.25889 是 embodiment 子带首篇 IT 级"不可能"工作；预计未来引发"adversarial channel capacity 应如何被实测/收紧"的方法论争论。
4. **mechanism vs verbal-interface 分离**：作为 yes/no 类 benchmark 的方法论必备控制项，正在从 causality 扩散到 reasoning、planning、theory-of-mind 等域。

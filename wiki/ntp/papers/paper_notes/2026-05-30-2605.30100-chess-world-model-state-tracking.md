# [arXiv:2605.30100] Chess-World-Model: A 10M-Game Benchmark for Exact State Tracking from Chess Move Sequences

- **Authors / Org**: Benjamin Walker, Terry Lyons (Oxford)
- **Published / Updated**: 2026-05-28
- **Primary cat / cats**: cs.LG
- **Link**: https://arxiv.org/abs/2605.30100

## 核心问题
world model 要求 *exact state tracking* (从动作序列维持正确 latent state); 现有 benchmark 多合成或纯语言, 缺少 *真实 domain + 严格 state 正确性*。Transformer 长期被理论与经验报告在 state-tracking 上弱于 input-dependent linear RNN, 是否在大规模真实 domain 上同型？

## 方法
10M 真实国际象棋对局, 任务 = 给定合法 move 序列预测确切 board state。两 split: held-out real-game + uniformly-random-legal OOD (测是否学到 transition rules vs 利用常见人类局面 shortcut)。同 interface / 同训练协议下对照 Causal Transformer / block-diagonal SLiCE / Mamba-3 / Gated DeltaNet (with negative eigenvalues)。

## 核心结论
- 3M / 8M 参数规模: recurrent 模型 *显著* 优于 Transformer。
- Real-game performance 在 ≥18M 参数饱和; 但 random-uniform OOD split 性能持续不饱和——多数模型在 shortcut 而非 transition rule 上学。
- 输入相关线性 RNN (Mamba-3 / Gated-DeltaNet w/ neg-eigenvalues) 在 state-tracking 上的形式优势在真实 domain 同型可观察, 与 Merrill 2404.08819 (illusion of state in SSMs) 间的 *negative-eigenvalue* / *expressive transition matrix* 区分线索一致。

## NTP 归类
- **类别**: NTP-mech (state-tracking 子带; 对 attention-only NTP 的实证上界, 对 SSM-w-neg-eig 的实证下界)
- **理由**: 在 10M 真实 game 规模 + matched protocol 下确认了 *attention-only NTP* 在精确 state-tracking 上的瓶颈, 同时给 SSM × expressive-transition-matrix 路线一个具体 evidence anchor; 直接喂养 §10 C-FORM-6 (SSM 作 TC⁰ 子集) / C-FORM-7 / C-FORM-8 (Looped-SSM 渐近塌缩) sub-candidate 的 *经验对偶*。

## 与已有工作的关系
- 与 §2 表 C-FORM-1 (TC⁰ wall) / Merrill 2404.08819 (SSM ⊂ TC⁰) 直接对照: SSM 形式上是 TC⁰ 子集但 *特定 transition matrix 选择* (negative eigenvalues + input-dependent) 在 state-tracking 任务族上给 attention-only NTP 没有的实证胜场。
- 与 §10 C-FORM-8 (Looped-SSM 渐近塌缩) 反向: C-FORM-8 预言 \"$T_{\\max}=O(1)$ Looped-SSM 与同 FLOPs MoE 等价\", 本文给 *single-step* SSM (无 loop) 已胜 Transformer, 提示 C-FORM-8 \"loop 是唯一旋钮\" 的命题在 *经验上* 太强——transition matrix 表达力本身也是旋钮, 需在 C-FORM-8 falsifier 中显式列出 \"matched transition-matrix expressivity\" 控制。
- 与 N2 §6 三道暗门 (latent CoT / unbounded rollout / tokenization) 互补: 加一条 *transition-matrix expressivity* 暗门, 把 N2 三暗门扩成四暗门候选 (后续 deepen tick 可写入 N2)。
- 与 [`topics/formal_limits.md`](../topics/formal_limits.md) §C-FORM-6 / §C-FORM-8 同步——本条目是 SSM × state-tracking 的 *首条 10M-scale + matched control* 实证。

## 是否提出新理论边界
no formal — 但提供可证伪经验命题: \"对任何 attention-only NTP, 存在参数规模无关的 exact state-tracking 上界 (Chess 18M 饱和 in-distribution / OOD 不饱和), 而 SSM-with-expressive-transition 不受同上界\"。配 Merrill 2404.08819 是该命题的形式骨架。

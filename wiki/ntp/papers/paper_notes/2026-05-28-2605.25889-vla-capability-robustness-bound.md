# [arXiv:2605.25889] Capability and Robustness Cannot Both Be Free: An Information-Theoretic Bound for Vision-Language-Action Models

- **Authors / Org**: anonymous (pre-print)
- **Published / Updated**: 2026-05-26
- **Primary cat / cats**: cs.CR (cross: cs.RO, cs.LG)
- **Link**: https://arxiv.org/abs/2605.25889

## 核心问题
VLA 模型在干净输入上 success > 95%，在 16/255 PGD 扰动下塌缩到 < 5%；现有经验防御以 clean accuracy 换 robustness，但是否存在一个 policy-independent 的 *理论* 上界，决定 capability + robustness 的总预算？

## 方法
对任何 discrete-action VLA policy，证明 **capability** (policy action 与 oracle action 的 MI) + **robustness** (对抗扰动下保留 MI，扣除 trivial channel leakage) ≤ task entropy + adversarial channel capacity。两次 Data Processing Inequality + MI 非负即得。pixel-level bound 在当前模型上松 (~10³ nats)；引入 encoder-specific corollary 把 channel 限制到 policy-relevant 子空间，OpenVLA 上预算从 ~5000 降到 ~31 nats。

## 核心结论
- 存在 policy-independent 的 capability+robustness 总预算；trade-off 不是 *工程偏好*，而是信息论硬约束。
- encoder-specific bound 已 close 到 actionable 数量级（policy 已经吃掉大部分预算），意味着 OpenVLA 类模型几乎不能"既高 capability 又高 adversarial robustness"。

## NTP 归类
- **类别**: NTP-mech 候选 (embodiment / VLA 子带的硬上界)
- **理由**: 给出 *与 policy 无关* 的信息论上界，符合"机制级不可能性"的形式要件。但适用范围限于 discrete-action policy + 给定 adversarial channel，未必直接迁移到 free-form text NTP。

## 是否提出新理论边界
yes — `MI(policy_action; oracle_action) + MI_preserved_under_adv ≤ H(task) + C(adversarial_channel)`；encoder-specific corollary 给 ~31 nats 数量级实测预算。

## Reviewer note (Xander)
最强反对：DPI 套两次确实给上界，但 "adversarial channel capacity" 这个项在 worst-case 下可以非常宽松，让 bound 几乎 vacuous；论文承认 pixel-level 松到 10³ nats，靠 encoder-specific corollary 才 actionable —— 这个 corollary 隐含\"扰动局限在 encoder 子空间\"假设，white-box 攻击者未必合作。作为 NTP-mech 候选纳入 *VLA 子带*，但**不能升格为通用 NTP-mech**。

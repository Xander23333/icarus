# [arXiv:2605.05438] On Semantic Loss Fine-Tuning Approach for Preventing Model Collapse in Causal Reasoning

- **Authors / Org**: cs.LG
- **Published / Updated**: 2026-05-06
- **Primary cat / cats**: cs.LG
- **Link**: https://arxiv.org/abs/2605.05438

## 核心问题
小模型 (Gemma 270M) 在 transitivity / d-separation 等 causal reasoning 微调任务上是否会出现**结构性 collapse**——学到与输入结构无关的"恒答 Yes/No"策略，并被高准确率掩盖？若是，单纯 cross-entropy NTP 微调能否避免？

## 方法
在 transitivity 与 d-separation 任务上用纯 CE 微调 Gemma 270M，观察 collapse 率与表观 accuracy；提出基于图逻辑约束的 semantic loss + 动态 λ 调度；对比 collapse 检测协议（混淆矩阵 + per-class accuracy）。

## 核心结论
- 纯 NTP CE 微调下 100% 实验出现 collapse；accuracy 仍可达 73.9% 但 per-class breakdown 揭示恒答策略。
- Semantic-loss + dynamic λ 把 collapse 率压到 0，transitivity 70.4% / d-separation 68.6%，且 per-class 平衡。
- 暗示 "NTP 不会因果推理" 的部分经验论据需排除 collapse confound——以 aggregate accuracy 评估的微调实验都需补 per-class / 分布漂移检查。

## NTP 归类
- **类别**: counter-evidence (针对 "NTP 学不会因果" 的方法论纠偏)
- **理由**: 把一类 NTP-mech 经验论据重写为 fine-tuning objective shaping 问题，证明"NTP 学不到因果结构"可能是 collapse-induced 假阳性。

## Reviewer note (Xander)
工作规模小（270M），易被忽略，但 collapse-by-CE 的现象在更大模型的因果 fine-tune 中是否存在尚未检验，**这正是值得我们复验的点**。最强反对：semantic loss 需要显式 graph constraints，等同于在 loss 中注入因果先验，本质上不再是纯 NTP 实验；用作"NTP 能学因果"的正面证据需谨慎。

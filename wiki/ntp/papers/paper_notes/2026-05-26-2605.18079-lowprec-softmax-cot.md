# [arXiv:2605.18079] The Expressive Power of Low Precision Softmax Transformers with (Summarized) Chain-of-Thought

- **Authors / Org**: Moritz Brösamle, Stephan Eckstein
- **Published / Updated**: 2026-05-18 / 2026-05-18
- **Primary cat / cats**: cs.LG / cs.LG, cs.CC, cs.CL
- **Link**: https://arxiv.org/abs/2605.18079

## 核心问题
此前 transformer expressivity 结果多依赖 hardmax / 高精度 / 不切实际的参数量，与真实模型脱节。能否在 **softmax + 有限精度 + 深宽 log-grow** 的现实条件下重新刻画 CoT 与 summarized CoT 的可计算性？

## 方法
先构造 ternary activation 的 hardmax transformer 模拟图灵机（标准 CoT），再通过良分离 attention scores 把它转成 softmax + 舍入的等价构造，避免病态参数；同一框架下分析 "summarized CoT"（中间步骤被压缩），证其相对原 CoT 更高效。最后在 Sudoku 上做 learnability 实验。

## 核心结论
- 在 softmax + 低精度 + depth/width = O(log T) 设定下，CoT transformer 可模拟图灵机 → 真实模型族在 CoT 加持下仍 Turing-complete。
- Summarized CoT 把规模需求从"时间 bound 对数"降到"空间 bound 对数"——压缩 trace 在表达力上更强。
- Sudoku 实验显示其 learnability 预测优于已有 high-precision expressivity 工作。

## NTP 归类
- **类别**: counter-evidence（对"NTP-mech depth horizon"派的部分软反驳）
- **理由**: 表明只要允许 CoT，"depth = 推理深度上界"的论断需要按 (CoT 长度 + 模型 depth) 双坐标讨论；纯 no-CoT 上界并不限定 NTP 模型的最终可计算能力。

## 与已有工作的关系
- 直接续 Pérez et al. (Turing completeness of Transformers)、Merrill & Sabharwal (TC⁰)；本文把"hardmax/高精度"假设拆掉。
- 与今日 2605.23024、2605.19944 形成张力：那两文给出 no-CoT 下的 architecture depth 天花板；本文说明 CoT 让 depth 可以被"token 维度时间"替换。
- 对 "summarized CoT" 与近年 latent reasoning / scratchpad compression 工作呼应。

## 是否提出新理论边界
yes (积极方向) — 现实参数下的 Turing-completeness 构造；以及 summarized CoT 的 "log space" 而非 "log time" 的紧致 size scaling。

## Reviewer note (Xander)
这是本批论文里方法最干净的一篇。它把 NTP-mech 阵营常用的 "TC⁰ depth bound" 论据范围限定为 *no-CoT / 固定 token 数*。讨论 NTP 的机制级上界时必须显式区分 "no-CoT NTP" 与 "CoT-augmented NTP"——后者基本与 TM 等价，不能轻易当 NTP-mech 候选。Sudoku 仅是 anecdotal evidence；需要更广 task suite 验证 learnability 预测。

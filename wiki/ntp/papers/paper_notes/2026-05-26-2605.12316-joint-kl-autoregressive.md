# [arXiv:2605.12316] Autoregressive Learning in Joint KL: Sharp Oracle Bounds and Lower Bounds

- **Authors / Org**: Yunbei Xu, Yuzhe Yuan, Ruohan Zhan
- **Published / Updated**: 2026-05-12 / 2026-05-12
- **Primary cat / cats**: cs.LG / cs.LG
- **Link**: https://arxiv.org/abs/2605.12316

## 核心问题
NTP 在 model misspecification 下的 long-horizon 学习理论：在 joint KL 这一与训练 log-loss 自然匹配的 metric 下，水平 H 对 approximation 与 estimation 两类 error 各自的真实依赖是什么？

## 方法
建立 joint KL 下 NTP 的 matching upper / lower bounds。对照 Hellinger 度量下已知的 Ω(H) approximation amplification，他们证明 joint KL 的 approximation factor 是 **horizon-free**；同时给出 Ω(H) 的信息论 estimation 下界，且对 decomposable 和 fully shared policy class 都成立，与 Õ(H) 上界匹配。

## 核心结论
- joint KL 下 approximation error 不随 H 放大；Hellinger 的 Ω(H) 是度量选择带来的伪 amplification。
- estimation error 的 Ω(H) 信息论下界与高效算法的 Õ(H) 上界紧匹配——长 horizon 学习的 estimation 代价是**信息论必然**，不是算法局限。
- joint-KL guarantee 可推出对 downstream task 的策略保证。

## NTP 归类
- **类别**: NTP-cap（NTP 学习理论的 horizon 复杂度结构）
- **理由**: 给出 NTP 在长序列下的 sharp 上下界；非"NTP 不可做某事"，而是"NTP 做长序列 imitation 至少要 Õ(H) 样本"。

## 与已有工作的关系
- 续 Foster 等关于 imitation learning / NTP 的 horizon dependence 工作；与 DAgger、behavior cloning 误差累积理论 (Ross & Bagnell) 对照。
- 与 Block et al. 关于 imitation learning 的 sharp bound 有承接。

## 是否提出新理论边界
yes — joint KL 下 NTP 估计误差有 Ω(H) 的信息论 lower bound，且匹配 Õ(H) 上界；approximation factor 是 horizon-free。

## Reviewer note (Xander)
理论很干净，是把"NTP 长序列模仿到底有多难"这件事钉死。结论解释了为何 RL/critic 在长 horizon 任务上常胜过 pure NTP——估计上界 Ω(H) 是 NTP 框架不可绕过的代价。是 NTP-cap 阵营本周最值得抄进 survey §6 长 horizon 那段的工作。

# [arXiv:2605.26379] When Does LeJEPA Learn a World Model?

- **Authors / Org**: David Klindt, Yann LeCun, Randall Balestriero
- **Published / Updated**: 2026-05-25
- **Primary cat / cats**: stat.ML (cs.LG)
- **Link**: https://arxiv.org/abs/2605.26379

## 核心问题
JEPA 一族（具体到 LeJEPA = alignment + Gaussian regularization）究竟在什么条件下学到的表征能 *线性* 恢复世界的潜变量？以及 Gaussian 这个先验是不是必须？

## 方法
- 假定世界由 latent 在 stationary、additive-noise 过渡下演化、被任意非线性 observation map 投影成观测；
- 在该生成假设下证明 LeJEPA 的 alignment 项+Gaussian 正则收敛到一个 spectral 解，使得 latent 可被 *linearly* identifiable (orthogonal up to permutation/sign)；
- 反向证明：在该类世界中，Gaussian 是 **唯一** 让该 guarantee 成立的 latent 先验（任何 non-Gaussian 备选都被排除）；
- 给出 approximate identifiability 的退化界；
- 验证：2D toy → 1024-d 隐空间 → pixel 级机器人控制 latent planning。

## 核心结论
- LeJEPA 在 stationary additive-noise 类世界上 *正面定理*：alignment + Gaussian reg → 线性可辨识；
- *唯一性定理*：Gaussian 是唯一具备该 guarantee 的 latent prior；
- 线性正交可辨识 ⇒ optimal latent-space planning；
- 实验：linear probe MSE 显著优于无正则 baseline；pixel control 收益与理论 degradation 界一致。

## NTP 归类
- **类别**: NTP-mech (世界模型 / 表示学习子带，paradigm-replacement 路线的 *理论加固*)。
- **理由**: 把 LeCun 的 JEPA "可证伪赌局" 从经验配方升级为 conditional 数学定理；同时给出 NTP 范式 *不会自动得到* 的性质（linear identifiability under nonlinear observation），构成 paradigm-level 分离的硬证据，但仅在论文给出的生成假设下成立。

## 是否提出新理论边界
yes — 条件性正面 + 唯一性双定理：
`若 latent 服从 stationary additive-noise dynamics、observation map 非线性可逆，则 LeJEPA 的 alignment + Gaussian regularization 是 linear identifiability 的 (必要充分) 唯一解之一。`

注意：定理依赖 (i) stationarity、(ii) additive Gaussian transition noise、(iii) Euclidean embedding 空间。SPHERE-JEPA 已展示在球面 manifold 上 (iii) 失败 → 结论不再适用。

## Reviewer note (Xander)
强结果是 *唯一性*：把"为何要 Gaussian"从工程直觉抬升到不可替代。但论文的世界假设 (stationary additive-noise) 几乎排除所有 *真实* embodied 环境（非平稳、状态相关噪声、流形结构）；与 SPHERE-JEPA 联读后，应把它定位为 "**理想化世界下 LeJEPA 是优解**" 的 *存在性* 证明，而非 NTP-vs-JEPA 通用结论。对 N6 §3 (world-model 三家答卷) 应作为 V-JEPA 路线的 *理论后盾* 引用，但同时标注其 stationarity 假设是 *待打破* 的前沿。

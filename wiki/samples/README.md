# Thesis 主线候选样章

7 个候选主线各一篇 ~4000 字样章，用来对比文风、定下三卷的主线 thesis，之后据此重写原 49 节。

每章下面按主线适配 4-5 个视角小节。

## 第一卷《架构》候选

- [**A — 为什么没人能逃离 Transformer**](A-no-escape-from-transformer.md)
  视角：编年史 / 逃亡者 / 胜利者的不安 / 混血儿
- [**B — scale 的三次危机与三次抢救**](B-three-crises-of-scale.md)
  视角：危机编年史 / 赌徒群像 / 被牺牲的路径 / 2026 的现在
- [**C — frontier 的分裂：从一条路变成五条路**](C-frontier-fragmentation.md)
  视角：分岔时刻 / reasoning / agentic / omni multimodal / open-weight

## 第三卷《评测》候选

- [**D — benchmark 的不可能三角**](D-impossible-triangle.md)
  视角：三角诅咒 / 三个尸体 / 塑造者两难 / 作弊者 / 评测者绝望
- [**E — 从 capability eval 到 agentic eval 的迁移**](E-capability-to-agentic-eval.md)
  视角：考题到沙盘 / 产品化 / 定义权之争 / 死亡名单

## 第二卷《可解释性》候选

- [**F — 从黑盒到神经科学的 60 年绕路**](F-60-years-to-neuroscience.md)
  视角：达特茅斯引子 / 两次黑盒化 / 三次失败起跳 / Olah 转向 / 现在
- [**G — 为什么 Anthropic 在可解释性上一家独大**](G-anthropic-monopoly.md)
  视角：人才地图 / OpenAI 失败 / DeepMind 路径 / 商业-安全飞轮 / 垄断代价
